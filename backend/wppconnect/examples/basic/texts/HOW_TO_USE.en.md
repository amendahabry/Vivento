## What is Vivento?

Vivento is an event management platform that helps you create, organize, and manage events with automated guest communication. Send WhatsApp invitations, track responses, collect photos, and manage everything from one intuitive dashboard.

## Common Tasks: Step-by-Step

### Task 1: Create an event

1. Sign in to your Vivento account
2. Your event is automatically created - click "Event Details" to edit
3. Enter event name, date, time, and location
4. Add coordinates for Google Maps/Waze integration (optional)
5. Add event notes or special instructions
6. Click "Save" to update event information

![Step 4 – Event details](docs/images/step4.png)

### Task 2: Add guests and send invitations

1. Go to "Upload Guest List" section
2. Choose one of three methods:
   - **Upload Excel/CSV:** Download template, fill in guest details, upload file
   - **Google Sheets:** Paste your Google Sheets link
   - **Manual entry:** Click "Add Guest Manually" and enter details one by one
3. Review your guest list
4. Select guests you want to invite
5. Click "Send invites" to send WhatsApp invitations

![Step 5 – Guest management](docs/images/step5.png)

### Task 3: Track responses and view analytics

1. Check the "Statistics" section for real-time RSVP data
2. View "Guest Response List" for detailed guest responses
3. Use the search function to find specific guests
4. Export data to Excel using the "Export to Excel" button
5. Monitor invitation status in the guest list

![Step 6 – Analytics dashboard](docs/images/step6.png)

### Task 4: Manage event photos

1. Navigate to "Event Photos" section
2. View all photos uploaded by guests in a grid layout
3. Click any photo to view it in full size
4. See photo details including guest name and upload date
5. Delete inappropriate photos if needed
6. Share the event link with guests so they can upload photos

![Step 7 – Photo gallery](docs/images/step7.png)

## Tips & Best Practices

- **Prepare your guest list in advance** using the provided Excel template for faster setup
- **Add event coordinates** to enable one-click navigation for guests via Google Maps/Waze
- **Send invitations in batches** rather than all at once to avoid overwhelming guests
- **Use the search function** to quickly find specific guests in large lists
- **Regularly check the photo gallery** to monitor guest uploads and remove inappropriate content
- **Export your data regularly** to keep backup copies of guest lists and responses
- **Test your event link** before sharing to ensure it works correctly
- **Set up your invitation image** in the dashboard to make invitations more personal

## Privacy & Data

Vivento stores your event data, guest lists, and uploaded photos securely. We use industry-standard encryption and never share your data with third parties. You can delete your account and all associated data at any time through your account settings. Photos are stored securely in the cloud and can be accessed only by you and your invited guests.

## Quick Troubleshooting

| Issue | Cause | Fix |
|-------|-------|-----|
| Can't sign in | Wrong credentials or account not verified | Check email/password, verify account via email link |
| Invitations not sending | Network issue or invalid phone numbers | Check internet connection, verify phone number format |
| Guest list upload failed | Wrong file format or missing required fields | Use provided Excel template, ensure all required columns are filled |
| Photos not loading | Network issue or expired links | Refresh the page, check internet connection |
| Event link doesn't work | Browser blocking or expired link | Try incognito mode, copy link manually, request new link |
| Can't delete photos | Permission issue or network problem | Refresh page, try again, contact support if persistent |
| Export to Excel fails | Browser issue or large file size | Try different browser, export smaller batches |
| Dashboard not loading | Browser cache or network issue | Clear browser cache, try different browser |
